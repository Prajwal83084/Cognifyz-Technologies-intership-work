Project Overview

This project focuses on building an end-to-end Insurance Analytics Dashboard to analyze business performance and enable data-driven decision-making.

The dashboard provides a centralized view of key metrics such as revenue, sales opportunities, renewals, and branch performance. It helps stakeholders monitor KPIs, identify growth opportunities, and improve overall business strategy.

 Objectives
Analyze insurance business performance
Track revenue, renewals, and sales pipeline
Evaluate branch and executive performance
Identify growth opportunities and risks
Build interactive dashboards for insights
 Dataset Description

The dataset includes multiple business components:

Policy Data (premium, renewals, policy type)
Opportunity Data (sales pipeline, deal stages)
Revenue Data (brokerage, invoices)
Account Executive Data (sales performance)
Branch Data (location-wise performance)
Meeting Data (sales activity tracking)

 Data Processing
 
Data imported from Excel files
Cleaned and formatted datasets
Handled missing values and inconsistencies
Created relationships between tables for analysis
 Tools & Technologies
Microsoft Excel
Power BI
Tableau
SQL
 Key KPIs
Total Brokerage
Total Open Opportunities
Stage Funnel Revenue
Target vs Achieved Revenue
Cross Sell / Renewal / New Business

Key Insights

High renewal rates ensure stable revenue flow
Cross-selling improves customer value and profitability
Low conversion in negotiation stages indicates follow-up gaps
Open opportunities reflect future revenue potential
Some branches underperform against targets

 Challenges Faced
 
Handling missing values
KPI formatting issues
Managing dynamic filters
Data relationship mismatches

 Recommendations
 
Improve renewal strategies for consistent revenue
Focus on high-value opportunity conversion
Increase digital adoption for policy renewals
Provide incentives to sales executives
Strengthen underperforming branches

 Business Impact
 
Improved customer retention
Increased revenue growth
Better sales productivity
Improved pipeline conversion rate
Enhanced forecasting accuracy

 Conclusion

This project demonstrates how data analytics and visualization tools can transform raw insurance data into meaningful insights. The dashboard enables real-time performance tracking and supports strategic business decisions.
